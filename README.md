# txmabc-django-cms

#### 介绍
基于Django开发的CMS框架，不定期持续更新！

#### 安装教程

1.  `git clone https://gitee.com/txmabc/txmabc.git`
2.  `cd txmabc`
3.  `docker build -t txmabc/txmabc .`
4.  `docker run --rm -d --name txmabc-django -p 8000:8000 -v ${PWD}:/code txmabc/txmabc`

#### 使用说明

1. 首页：http://127.0.0.1:8000
